# Sentrix Command Center

### Turning Security Events into Actionable Intelligence

Sentrix Command Center is a next-generation Mini Security Operations Center (SOC) platform designed to simulate the complete cyber defense lifecycle.

## Architecture

Kali Linux (Attacker)
↓
DVWA (Target)
↓
Suricata IDS
↓
Wazuh SIEM
↓
Keycloak IAM
↓
Shuffle SOAR
↓
Sentrix Command Center Dashboard

## Features

- Attack Simulation using Kali Linux
- Intrusion Detection with Suricata
- Security Monitoring via Wazuh
- Identity & Access Management with Keycloak
- Automated Response using Shuffle
- Threat Scoring Engine
- Attack Timeline Replay
- Executive Risk Dashboard
- SOC Analyst View
